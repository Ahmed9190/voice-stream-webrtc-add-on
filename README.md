# Home Assistant Add-on Repository: Hello World PoC

This is a proof-of-concept repository containing a "Hello World" add-on.

## Installation

1.  Navigate to your Home Assistant instance.
2.  Go to **Settings** -> **Add-ons** -> **Add-on Store**.
3.  Click the three dots in the top right corner and select **Repositories**.
4.  Add the URL of this GitHub repository.
5.  Refresh the store (if needed), find "Hello World", and install it.
